Projects

Project Title : Network Management Tool

Period : June-05 to June-06

Team Size : 3

Role : Core Team Member

Environment : Windows 98/2000/XP, LAN, Java, Java RMI, Java Swings

Domain : Networking

The project was executed at our College campus with a group of young, enthusiast and dynamic team of college students. The goal of the project was to develop a software tool for Network Administration, which can be done remotely. My contribution to the team was to gather requirements by conducting meeting with our college’s Network Administrator and come up with a list of items, which could facilitate him to manage network resources efficiently. With this we came up with a requirements specification and started working on the design. We then did research of which technology to be used and decided upon Java. I wrote Java programs using Java’s RMI technology.
