Final Year ME Projects

Project Title :

Effective Information Retrieval System

Period :

August 2011- Decemeber 2013

Environment :

Windows XP, LAN, Java

Domain :

Data Mining and Information Retrieval


1. Problem Definition

Implementation of GA based Information Retrieval System using Singular Value Decomposition and Principle Component Analysis techniques.


2. Objectives

To produce the reduced dataset by singular value decomposition (SVD) and Principal Component Analysis (PCA).

To improve the retrieval efficiency by reducing the information retrieval time by using Genetic Algorithm.

To achieve accuracy in information retrieval results using Genetic Algorithm.


3. Scope

The project was only used for query based, domain specific text documents and not for web pages (web mining).

Dimensionality reduction techniques like Singular Value Decomposition (SVD) and Principle Component Analysis (PCA) were used to produce reduced dataset.

Genetic Algorithm is one of the powerful searching mechanism known for its robustness and quick search capabilities. So, it was used for with Fitness Functions for fast and effective information retrieval.
